# LennySkill

Product management decision-making skill for Cursor/Claude, inspired by practical frameworks commonly discussed in Lenny's Podcast.

## What this repository contains

- `.cursor/skills/product-management-decision-process/SKILL.md`
- `.cursor/skills/product-management-decision-process/reference.md`

The skill provides:

- A structured decision workflow (frame -> criteria -> options -> recommendation -> validation)
- Deep playbooks for:
  - Product-Market Fit evaluation
  - North Star Metric framework
  - User retention strategies
  - Feature prioritization methods
  - Growth loops
  - Activation funnels

## How to use

1. Copy the `.cursor/skills/product-management-decision-process/` folder into a project that uses Cursor skills.
2. Ask for product strategy or prioritization help (for example: "evaluate PMF", "define a North Star metric", "prioritize features", "improve activation funnel").
3. The skill will apply the decision template and relevant deep framework sections.
